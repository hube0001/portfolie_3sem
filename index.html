Opskrift App

I dette projekt byggede jeg en opskrift-app i React, hvor jeg brugte useState til at håndtere applikationens state — herunder opskrifternes navn, ingredienser, instruktioner og tider. Hver gang brugeren tilføjede en ny ingrediens, opdaterede React automatisk UI'et uden at genindlæse siden, hvilket gav mig en konkret forståelse for hvordan reaktiv state fungerer i praksis. Projektet lærte mig også at tænke i komponenter — jeg opdelte appen i mindre dele som et formular-komponent og et liste-komponent, så koden blev mere overskuelig og genanvendelig.

Hvad jeg lærte
  • useState til at holde styr på dynamiske inputfelter og opskriftdata
  • Komponentopdeling — formular og liste som separate komponenter

FamilyFriends App

En mobilapp til hundeejere bygget i Next.js med React og Tailwind CSS — med raceoversigt, detaljerede hundeprofiler, favoritsystem og søgefunktion.   • Datahentning med Next.js Server Components
  • Persistering af favoritter med localStorage
  • Søgefunktion med URL search params i Next.js
  • Komponentbaseret UI med React og Tailwind CSS
